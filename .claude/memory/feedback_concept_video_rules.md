Search YouTube video FIRST. Concept name comes from what the video actually covers. Never write concept names first and then search for a matching video.

**Why:** Naming concepts before video search causes mismatched content — the video covers something slightly different, or one video covers multiple concepts but only one is listed.

## Process
1. Search YouTube for the topic (e.g. "pandas groupby tutorial")
2. Pick the best video (recent 2022+, focused, trusted channel)
3. Concept name = what that video covers (e.g. "GroupBy and Aggregation in Pandas")
4. If one video covers 2 tightly related things → create both concepts mapped to that video
5. No video found → still include the concept, set videoUrl/videoTitle to null
6. Verify: recent (2022+), focused (not 4-hour full courses), trusted channel
